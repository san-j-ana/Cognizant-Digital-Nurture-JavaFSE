# Exercise 1: Logging Error Messages and Warning Levels

## Objective
To demonstrate logging warning and error messages using SLF4J and Logback.

## Files
- pom.xml
- LoggingExample.java

## Concepts Used
- SLF4J
- Logback
- Logger
- Warning Log
- Error Log

## Description
- Added SLF4J and Logback dependencies.
- Created a logger using `LoggerFactory`.
- Logged one warning message.
- Logged one error message.

## Expected Output

WARN  - This is a warning message

ERROR - This is an error message

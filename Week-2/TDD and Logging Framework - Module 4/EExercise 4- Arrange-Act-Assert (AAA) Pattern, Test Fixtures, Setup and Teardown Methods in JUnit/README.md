# Exercise 4: Arrange-Act-Assert (AAA) Pattern in JUnit

## Objective
To organize JUnit test cases using the Arrange-Act-Assert (AAA) pattern and implement setup and teardown methods using `@Before` and `@After`.

## Files
- Calculator.java
- CalculatorTest.java

## Concepts Used
- Arrange-Act-Assert (AAA) Pattern
- @Before
- @After
- @Test
- assertEquals()

## Description

### Arrange
Initialize the required objects and input values.

### Act
Call the method to be tested.

### Assert
Verify the expected result using JUnit assertions.

### @Before
Runs before every test case and creates the required object.

### @After
Runs after every test case and performs cleanup.
